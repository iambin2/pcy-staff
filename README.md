# 포켓몬 센터 연세점 · 임원진 업무실 (1단계: 로그인)

공유 아이디·비밀번호로 임원진만 들어올 수 있는 사이트의 뼈대입니다.
비밀번호 확인은 '문지기(Apps Script)'가 하므로 안전합니다.
프로그래밍 지식 없이, 아래 3단계만 따라 하면 됩니다.

---

## 1. 데이터 시트 + 아이디/비밀번호 만들기

1. 클럽 구글 계정으로 새 구글 스프레드시트 생성 (이름: 예 `포센연 임원진 데이터`)
2. 시트 탭 이름을 **`설정`** 으로 변경
3. 아래처럼 입력합니다. (A열=항목, B열=값. 1행은 제목)

| 항목(A) | 값(B) |
|---|---|
| 아이디 | pokecenter |
| 비밀번호 | (길고 추측하기 어려운 비밀번호) |
| 비밀키 | (아무 랜덤 문자열, 한 번 정하고 그대로 두기) |
| 표시이름 | 임원진 |
| 역할 | 공용 계정 |

> 나중에 비밀번호를 바꾸고 싶으면 이 칸만 수정하면 됩니다.

## 2. 문지기 설치하기 (Apps Script)

1. 그 스프레드시트에서 `확장 프로그램` → `Apps Script`
2. 기본 코드를 지우고, 이 프로젝트의 `apps-script/Code.gs` 내용을 전부 붙여넣기 → 저장
3. 오른쪽 위 `배포` → `새 배포` → 유형: **웹 앱**
   - 실행: **나**
   - 액세스 권한: **모든 사용자**
   - 배포 → 권한 승인(처음 1회) → 나오는 **웹 앱 URL** 복사 (`.../exec`)

> 코드나 설정을 바꾼 뒤에는 `배포 > 배포 관리 > 편집(연필) > 버전: 새 버전 > 배포` 를 해야 반영됩니다.

## 3. 사이트 올리기 (GitHub)

1. `src/config.js` 를 열어 `APPS_SCRIPT_URL` 을 2단계에서 복사한 URL로 교체
2. GitHub에서 새 저장소 생성 후, 이 폴더 전체를 업로드 (node_modules, dist 제외)
3. 저장소 `Settings` → `Pages` → Source를 **GitHub Actions**로 선택
4. 잠시 뒤 `Actions` 탭에서 배포가 끝나면 `https://내아이디.github.io/저장소이름/` 으로 접속

---

## 로컬에서 미리보기 (선택)

```bash
npm install
npm run dev
```

## 잘 안 될 때

- **로그인 후 "연결 문제"**: `APPS_SCRIPT_URL`이 `.../exec`로 끝나는지, 웹 앱 액세스가 "모든 사용자"인지 확인
- **비번이 맞는데 안 됨**: `설정` 시트를 고쳤다면 Apps Script를 "새 버전"으로 다시 배포했는지 확인
- **시트를 찾을 수 없음 오류**: 시트 탭 이름이 정확히 `설정` 인지 확인
